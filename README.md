# Phonebook
[Java EE] 연락처 앱 프로젝트

### 프로젝트 기간
+ 2023-04-18 ~ 2023-04-29

### 기술스택
+ Java EE,Java1.8,JSP,JSTL,myBatis,H2,JavaScript

### 기능
+ 쿠키를 이용한 CSS 테마 변경 
+ Session을 이용한 이름 바인딩
+ BCrypt를 사용한 비밀번호 암호화
+ 로그인 , 회원가입 유효성 검사 
+ WebFilter를 사용하여 로그인유저에게만 사이트 특정 권한부여 
+ 연락처 추가,수정,삭제
