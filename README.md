# 📪  Phonebook Application 📞
<p>Java EE, myBatis, H2 </p>

<h1>💭연락처 프로젝트 구상 및 접근방법<h1>

<h3>프로젝트명</h3>
 
<h4>Phonebook Application</h4>

<h3>프로젝트 개요</h3>
<p>사용자는 웹사이트 회원가입 후, 친구목록에 연락처를 추가할 수 있음</p>
<p>추가된 연락처를 한번에 볼 수 있는 목록이 있으며, 연락처 삭제 가능</p>
